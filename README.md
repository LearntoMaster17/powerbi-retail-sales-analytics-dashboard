# SG Mart Dashboard

A comprehensive Power BI dashboard project for SG Mart, designed to deliver actionable business insights on sales, profit, customer segmentation, and shipping performance across multiple European regions. This project demonstrates advanced data visualization, business analytics, and storytelling skills using Power BI.

---

## Project Overview

The **SG Mart Dashboard** provides a holistic view of sales and operational performance for a retail business operating across Europe. The dashboard is built using Power BI and leverages data from Excel sources to create interactive, insightful, and visually appealing reports for business stakeholders.

---

## Key Features

- **Executive Summary KPIs**:  
  - Total Sales, Total Profit, Total Quantity Sold, and Total Orders at a glance.
- **Sales Analysis**:  
  - Sales and profit breakdown by category and sub-category.
  - Top-performing products and customer segments.
- **Customer Insights**:  
  - Customer base and sales distribution by country.
  - Detailed customer drill-down pages (e.g., order history, returns, and product preferences).
- **Shipping Performance**:  
  - Analysis of shipping modes used and their impact on order fulfillment.
- **Geographical Analysis**:  
  - Sales mapped by country and state, with interactive visuals for regional performance.
- **Time Series Trends**:  
  - Quarterly trends for sales and profit, highlighting seasonality and growth patterns.
- **Actionable Insights**:  
  - Automated insights and recommendations based on data patterns (see dashboard notes).

## Files Included

- **SG Mart Dashboard.pbix**  
  The main Power BI dashboard file containing all reports, visuals, and data model.
- **SG Mart Dashboard.xlsx**  
  The source data file used for building the dashboard (imported into Power BI).
- **SG Mart Dashboard.pdf**  
  Exported PDF version of the dashboard for quick reference and sharing.
- **SG Mart Dashboard_page-0001.jpg ... page-0004.jpg**  
  Image snapshots of key dashboard pages for preview and documentation.

---

## How to Use

1. **Open the Dashboard**  
   - Launch Power BI Desktop.
   - Open `SG Mart Dashboard.pbix` to explore and interact with the full dashboard.

2. **Review the Data**  
   - The dashboard is powered by `SG Mart Dashboard.xlsx`. You can review or update the data as needed.

3. **Export & Share**  
   - Use the PDF and image files for sharing insights with stakeholders who do not have Power BI access.

---

## Insights & Highlights

- **Top Sales Regions**: France and Germany lead in both customer base and sales contribution.
- **Best Performing Categories**: Bookcases and Copiers are top sub-categories in their respective segments.
- **Shipping Optimization**: Economy shipping is the most utilized, accounting for ~60% of orders.
- **Customer Drilldown**: Individual customer pages provide detailed order and return history for personalized analysis.

## Getting Started

To run or modify this project:
1. Download all files in the `powerbi-retail-sales-analytics-dashboard` folder.
2. Open the `.pbix` file in Power BI Desktop (free download from Microsoft).
3. Refresh the data if you update the `.xlsx` file.
4. Explore, analyze, and customize the dashboard as needed.
